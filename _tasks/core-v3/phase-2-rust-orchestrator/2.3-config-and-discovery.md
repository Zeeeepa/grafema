# Task 2.3: Config and Discovery

## Goal

Parse YAML config (`grafema.config.yaml`), discover source files via glob patterns respecting `.gitignore`.

## Files to Modify

- `packages/grafema-orchestrator/src/config.rs`
- `packages/grafema-orchestrator/src/discovery.rs`

## Subtasks

### 2.3a: Config structs
```rust
#[derive(Debug, Deserialize)]
pub struct AnalyzerConfig {
    pub root: PathBuf,
    pub include: Vec<String>,      // glob patterns: ["src/**/*.js", "src/**/*.ts"]
    pub exclude: Vec<String>,      // glob patterns: ["node_modules/**", "dist/**"]
    pub plugins: Vec<PluginConfig>,
    pub rfdb_socket: Option<PathBuf>,
}
```

### 2.3b: Plugin config
```rust
#[derive(Debug, Deserialize)]
pub struct PluginConfig {
    pub name: String,
    pub command: String,           // e.g., "grafema-resolve imports"
    pub query: Option<String>,     // Datalog query for streaming mode input
    pub depends_on: Vec<String>,   // plugin names this depends on
    pub mode: PluginMode,          // Streaming | Batch
    pub timeout: Option<Duration>,
}

#[derive(Debug, Deserialize)]
pub enum PluginMode {
    Streaming,  // orchestrator queries RFDB, pipes to plugin stdin
    Batch,      // plugin gets RFDB_SOCKET env var, queries directly
}
```

### 2.3c: File discovery
- Walk `config.root` directory
- Match files against `include` glob patterns
- Filter out files matching `exclude` patterns
- Return sorted `Vec<PathBuf>` of discovered files

### 2.3d: Respect .gitignore
- Use the `ignore` crate for .gitignore-aware walking
- Layer: .gitignore rules → config excludes → config includes
- Handle nested .gitignore files

## Acceptance Criteria

- Given `include: ["src/**/*.js"]`, correctly discovers all matching files
- Excluded patterns are respected
- .gitignore'd files are skipped
- Invalid config produces clear error messages
- Empty include patterns → error (not silent empty list)

## Dependencies

- **2.1** (crate must exist)

## Estimated Scope

~0.5 day. Straightforward config parsing + file walking with the `ignore` crate.
