### 1.6b: Test helpers
`analyzeJS :: Text -> FileAnalysis` — helper that:
1. Takes JS source code as Text
2. Shells out to OXC to parse into ESTree JSON (or uses pre-generated fixture JSON)
3. Runs analyzer on the AST
4. Returns FileAnalysis for assertions

Alternative: fixture-based approach with pre-generated OXC JSON files in `test/fixtures/`.

### 1.6c: Scope resolution tests
- Variable reference: `const x = 1; console.log(x);` → READS_FROM edge
- Function call: `function f() {} f();` → CALLS edge
- Cross-scope: `const x = 1; function f() { return x; }` → READS_FROM edge (closure)
- Block scope: `{ let x = 1; } console.log(x);` → x unresolved outside block
- Nested scopes: inner declaration shadows outer

### 1.6d: Hoisting tests
- `var` hoisting: `function f() { x = 1; var x; }` → x resolves
- `let`/`const` TDZ: `{ console.log(x); let x = 1; }` → x should NOT resolve (TDZ)
- Function declaration hoisting: `f(); function f() {}` → CALLS edge resolves
- `var` in block: `if (true) { var x = 1; } console.log(x);` → x resolves (function-scoped)

### 1.6e: Export tests
- Named export: `export function foo() {}` → ExportInfo(name="foo", kind=named)
- Default export: `export default class Bar {}` → ExportInfo(name="default", kind=default)
- Re-export: `export { baz } from './qux'` → ExportInfo(kind=reexport, source="./qux")
- Star re-export: `export * from './utils'` → ExportInfo(name="*", kind=reexport)

### 1.6f: Edge case tests
- Shadowing: `const x = 1; function f() { const x = 2; return x; }` → inner x
- Closures: `function outer() { const x = 1; return () => x; }` → closure captures x
- Class methods: `class C { method() { this.field; } }` → appropriate references
- Destructuring: `const { a, b } = obj; console.log(a);` → a resolves
- Arrow functions: `const f = (x) => x + 1;` → parameter resolves
- Catch binding: `try {} catch (e) { console.log(e); }` → e resolves in catch scope

### 1.6g: Golden tests
Snapshot comparison of full FileAnalysis for fixture files:
- `test/fixtures/basic-scope.js` → `test/golden/basic-scope.json`
- `test/fixtures/imports.js` → `test/golden/imports.json`
- `test/fixtures/classes.js` → `test/golden/classes.json`
- Compare full JSON output to detect regressions
